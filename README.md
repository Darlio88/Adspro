# AdsPro (Classified Ads Platform)

## Description

This is a classified ads platform built using Next.js. It provides a user-friendly interface for users to post and browse classified ads across different categories. The platform allows users to create accounts, post ads, search for ads, and contact advertisers.

## Features

- User registration and authentication system
- Create, edit, and delete classified ads
- Search and filter ads by category, location, and keywords
- Messaging system for users to contact advertisers
- Responsive design for optimal viewing on different devices
- Admin panel to manage users and ads
- Secure payment integration for premium ads (optional)

## Technologies Used

- Next.js: A React framework for server-side rendering and building web applications.
- React: A JavaScript library for building user interfaces.
- MongoDB: A NoSQL database for storing user accounts, ads, and other data.
- Express.js: A minimal and flexible web application framework for Node.js.
- Tailwind CSS: A utility-first CSS framework for styling the UI components.
- SendGrid: An email delivery platform for sending email notifications.
- Stripe: A payment processing platform for accepting online payments (optional).

## Installation

1. Clone the repository:

   ```
   git clone https://github.com/darlio88/AdsPro.git
   ```

2. Install the dependencies:

   ```
   cd AdsPro
   npm install
   ```

3. Set up the environment variables:

   - Rename the `.env.example` file to `.env`.
   - Replace the placeholder values in the `.env` file with your own configuration (e.g., database connection string, API keys).

4. Run the development server:

   ```
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:3000` to access the platform.

## Deployment

To deploy the platform to a production environment, you can follow the deployment guides for popular hosting platforms like Vercel, Heroku, or AWS. Make sure to set up the necessary environment variables in your production environment.
